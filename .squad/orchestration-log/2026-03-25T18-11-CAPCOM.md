- CLI 1.0.5: Embedding-based dynamic skill instruction injection per turn

**Key insight:** SDK version pinning (^0.1.32) only controls the SDK adapter. The CLI runtime (system prompt assembly, agent lifecycle) auto-updates independently. Squad needs platform-resilient prompt injection.

**Recommended actions:**
- Clean up template file naming to prevent duplicate discovery
- Upgrade to SDK 0.2.0 "customize" mode for resilient prompt control
- File CLI issue requesting instruction discovery exclusions and idle timeout config options
